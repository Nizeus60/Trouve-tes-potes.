# 📍 Où est mon pote — notice

App Android pour se retrouver en festival : flèche + distance + vibration.

## Installer (sur CHAQUE téléphone)
1. Envoyer le fichier `OuEstMonPote.apk` sur le téléphone
   (câble USB, e-mail, Google Drive, Telegram… ⚠️ WhatsApp bloque les .apk).
2. Ouvrir le fichier depuis le téléphone → Android demande
   « Autoriser l'installation d'apps inconnues » → **Autoriser**.
3. Installer, ouvrir l'app, **accepter la localisation** (« Pendant l'utilisation »)
   **ET les appareils à proximité / Bluetooth**, puis **activer le Bluetooth** du tel.

## Utiliser
1. Chacun met son **prénom**.
2. **Le même code de groupe** pour tout le monde (ex : `LICORNE42`). Le 🎲 en génère un.
3. « C'est parti 🚀 ».
4. La grosse flèche pointe vers le pote le plus proche, la distance s'affiche,
   ça vibre/bip de plus en plus vite quand tu te rapproches.
5. Plusieurs potes ? Ils s'affichent en liste, tape sur un nom pour le suivre.

## Les 2 modes (automatiques)
- **GPS** = mode principal, pour se rapprocher de loin (flèche + distance).
- **Bluetooth** = prend le relais sur les **derniers mètres** : une barre
  « 🔥 tu chauffes / ❄️ tu refroidis » et la vibration s'emballe quand le
  signal monte. Pas de direction, juste « plus chaud/plus froid ».
  ⚠️ Bruité en foule (les corps bloquent le signal) — c'est normal.
  Garde le **Bluetooth activé** sur les deux tels.

## Menu options ⚙️ (en haut à droite)
- 🔊 **Son** : coupé par défaut (réactivable ici).
- 📳 **Vibration** : activée par défaut.
- 🎚️ **Sensibilité** : plus haut = vibre/bip plus tôt et plus vite.

## Bon à savoir
- Marche à **plusieurs** (3, 4, 5… potes), même code pour tous.
- **Flèche** : beaucoup de **tablettes n'ont pas de boussole** → la flèche ne
  peut pas s'orienter à l'arrêt. Dans ce cas **marche quelques pas** : l'app
  déduit ta direction de ton déplacement. Sur un téléphone avec boussole,
  c'est instantané.
- Tout proche, le GPS devient flou : la flèche **se fige sur la dernière
  direction connue** et la barre Bluetooth « tu chauffes » prend le relais.
- La flèche utilise la **boussole** : si elle ne s'oriente pas, fais un 8 avec
  le tel ou marche quelques mètres.
- Précision GPS ≈ 5–15 m : ça t'amène « à vue », pas pile sur la personne.
- Les positions passent par un **serveur public gratuit**. Pour la discrétion,
  utilise un code de groupe pas évident.
- Garde l'app **ouverte à l'écran** pour le suivi temps réel.
